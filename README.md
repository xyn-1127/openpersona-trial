# OpenPersona — replication

A small, self-contained replication of the OpenPersona persona-creation
walkthrough: write a persona config, generate the persona, then chat with it
or run a short survey through it.

The persona files are generated by the `openpersona` toolkit. The chat and
survey scripts here are small Ollama-based equivalents.

## Prerequisites

- Node.js 18+
- [Ollama](https://ollama.com) running locally, with the model pulled:

  ```bash
  ollama pull qwen3.5:9b
  ollama serve
  ```

## Quick start

```bash
# 1. Install the pinned OpenPersona version from package-lock.json
npm ci

# 2. Generate the two example personas
npx openpersona create --config ./rosa.json
npx openpersona create --config ./maya.json

# 3. Chat with either generated persona
node chat.js rosa-mendez
node chat.js maya-santos

# 4. Run a short survey through either persona (saves a CSV)
node survey-chat.js rosa-mendez
node survey-chat.js maya-santos
```

## What's in here

| File | What it is |
|------|------------|
| `rosa.json` | A persona config (static — no evolution) |
| `maya.json` | A persona config with evolution enabled |
| `chat.js` | Interactive chat with a generated persona (Ollama backend) |
| `survey-chat.js` | Asks a persona a list of questions, saves answers to a CSV |
